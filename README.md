# 燕京理工学院 招生助手

河北省高考一分一档查询工具，支持 2025/2026 两年数据，物理类/历史类切换。

## 功能

- 输入分数查位次（2025 + 2026 双年对比）
- 按位次反查等效分
- 55个本科/专科专业录取参考
- 自动匹配可报考专业

## 部署

纯静态页面，直接部署到 Render / Vercel / Netlify 即可。

### Render 部署步骤

1. 连接 GitHub 仓库
2. 新建 Static Site
3. 分支：main
4. 构建命令：空（纯静态）
5. 发布目录：`/`

## 本地使用

直接双击 `index.html` 即可，无需服务器。
